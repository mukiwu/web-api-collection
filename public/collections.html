Intersection Observer API

用於監測元素進入或離開視窗的 API，可用於實現無限滾動、延遲載入圖片或影片等功能。

瀏覽器相容性

Chrome 51+
Firefox 55+
Safari 12.1+
Edge 15+
Opera 38+

程式碼範例 

// 建立 Intersection Observer
const observer = new IntersectionObserver(entries => {
for (const entry of entries) {
if (entry.isIntersecting) {
// 元素進入視窗
entry.target.classList.add('visible');
} else {
// 元素離開視窗
entry.target.classList.remove('visible');
}
}
}, {
root: null,  // 使用視窗作為參考
threshold: 0.1  // 當元素 10% 可見時觸發
});
// 監測所有需要的元素
const elements = document.querySelectorAll('.lazy-load');
elements.forEach(el => observer.observe(el));

Web Speech API

讓網頁應用能夠進行語音辨識和語音合成的 API，可用於語音指令、聲控介面或無障礙功能。

瀏覽器相容性

Chrome 33+
Firefox 49+
Safari 14.1+
Edge 79+
Opera 19+

程式碼範例 

// 語音合成範例
function speak(text) {
const utterance = new SpeechSynthesisUtterance(text);
utterance.lang = 'zh-TW';
utterance.rate = 1.0;
utterance.pitch = 1.0;
// 選擇語音
const voices = speechSynthesis.getVoices();
const voice = voices.find(v =>
v.lang.includes('zh') && v.name.includes('Female')
);
if (voice) utterance.voice = voice;
// 開始說話
speechSynthesis.speak(utterance);
}

Web Storage API

提供在瀏覽器中儲存資料的機制，包含 localStorage（永久儲存）和 sessionStorage（暫時儲存）兩種方式。

瀏覽器相容性

Chrome 4+
Firefox 3.5+
Safari 4+
Edge 12+
Opera 10.5+

程式碼範例 

// localStorage 使用範例
// 儲存資料
localStorage.setItem('username', '張小明');
localStorage.setItem('theme', 'dark');
// 讀取資料
const username = localStorage.getItem('username');
const theme = localStorage.getItem('theme');
// 刪除特定資料
localStorage.removeItem('theme');

Geolocation API

允許網頁獲取使用者的地理位置資訊，可用於地圖應用、位置相關服務或本地化內容。

瀏覽器相容性

Chrome 5+
Firefox 3.5+
Safari 5+
Edge 12+
Opera 10.6+

程式碼範例 

// 獲取使用者位置
function getLocation() {
if (navigator.geolocation) {
navigator.geolocation.getCurrentPosition(
// 成功回調
position => {
const latitude = position.coords.latitude;
const longitude = position.coords.longitude;
const accuracy = position.coords.accuracy;
console.log(`位置: ${latitude}, ${longitude}`);
console.log(`精確度: ${accuracy} 公尺`);
// 使用位置資訊...
},
// 錯誤回調
error => {
switch(error.code) {
case error.PERMISSION_DENIED:
console.error("使用者拒絕位置請求");
break;
case error.POSITION_UNAVAILABLE:
console.error("位置資訊不可用");
break;
case error.TIMEOUT:
console.error("位置請求逾時");
break;
}
},
// 選項
{
enableHighAccuracy: true,  // 高精度
timeout: 5000,  // 5秒逾時
maximumAge: 0  // 不使用快取
}
);
} else {
console.error("瀏覽器不支援地理位置");
}
}
